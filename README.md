# 🔢 Number Guessing Game

A console-based number guessing game built with C#. The program generates a random number, and the player tries to guess it while receiving hints until the correct answer is found.


---

## 🚀 Features
- 🎲 Random number generation
- 🔼 Hints for higher guesses
- 🔽 Hints for lower guesses
- 🔁 Multiple attempts
- 📥 User input handling

---
  

## 🛠️ Built With
- C#
- .NET
